# 🦩 Qomer POS - La Riviera
**Sistema de Gestión Integral para Hostelería (Carta Digital + TPV + Cocina)**

Este proyecto es una solución de software real desarrollada como Trabajo de Fin de Grado (TFG). Diseñado para automatizar y mejorar la experiencia de usuario en el bar-restaurante y piscinas "La Riviera", permitiendo a los clientes escanear la carta, a los camareros tomar notas en tiempo real y a la cocina gestionar los pedidos.

### 🚀 Stack Tecnológico
* **Frontend:** React, React Router, Vite
* **Estilos:** Tailwind CSS, DaisyUI
* **Backend & Base de Datos:** Supabase (PostgreSQL, Realtime, Auth)
